# Roadmap

## MVP

- Manual purchase registration
- Return, refund, and warranty date management
- Dashboard for urgent deadlines
- Receipt image/PDF attachments
- CSV/JSON export
- Sample data

## v1+ Candidates

- OCR extraction for receipt date, merchant, and amount
- Calendar `.ics` export
- PWA install support
- Local encryption option
- Product manuals, serial numbers, and repair history
- Full backup export including receipt files
- Import from JSON backup
- Configurable reminder thresholds
- Browser notification reminders
- Merchant/category filters
